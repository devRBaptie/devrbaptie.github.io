FlappyBird Game

Created an Android Mobile Game in Java. The objective of the game is to get the highest score possible. Using bitmap to use a image as a character and canvas to create objects that move, tap the screen to move the bird and catch the blue ball to earn point and avoid the black dots else you lose a life. Multiple Choice Quiz

C# WinForms desktop application which allows users and lecturers to login via mySQL. Once a lecturer is logged in they may add a new Multiple Choice Quiz (MCQ) but entering a question and 2 wrong answers and the correct answer. When a MCQ has been completed by a student the lecturer will be able to view their marks. Once a student has logged in, if there is an avalible MCQ they can complete the quiz and once they have completed it they will be able to view their mark. All data is stored in a mySQL DB.
